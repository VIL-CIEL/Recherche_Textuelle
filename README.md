# Recherche_Textuelle
Projet académique de lycée en première NSI

## - Comment ouvrir les fichiers sources ?
- Télécharger la dernière release<br>
- Dézipper le fichier<br>
- Ouvrez le dossier 'Recherche_Textuelle-1.0.0'<br>
- Ouvrez le fichier Interface_Utilisateur.py avec un IDE de code avec python et la dernière version de pygame installé<br>
- Cliquez sur le bouton play (en haut à droite de la fenetre pour VS Code)

Et voici le fichier source complet !

## - Utilisation :
Vous avez deux onglets en haut a gauche : File permet de changer de base de données en .csv, Algorithme permet de changer le type d'algorithme utilisé<br>
Puis vous avez juste a rentrer le mot que vous voulez chercher puis appuyer sur le bouton chercher ! Toutes les occurences seront listées à droite
